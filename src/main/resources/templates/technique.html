Document Technique

Spécifications techniques de la plateforme Atizay, basée sur une architecture robuste et moderne.

Stack Technologique

Backend Java 21 avec Spring Boot 3.4.1. Gestion de la sécurité via Spring Security et OAuth2.
Frontend Thymeleaf pour le templating server-side, HTML5, CSS3 (Vanilla) et JavaScript moderne (ES6+).
Base de Données MySQL 8.0 pour le stockage persistant, géré via Spring Data JPA (Hibernate).

Architecture Logicielle

Atizay suit le pattern MVC (Modèle-Vue-Contrôleur) pour assurer une séparation claire des responsabilités.

- model: Entités JPA (Salon, Client, Prestation, etc.)
- repository: Interfaces Spring Data JPA pour l'accès aux données
- service: Logique métier et orchestrations
- controller: Endpoints web et gestion des requêtes
- config: Paramétrage de la sécurité et de l'application

Gestion des Médias

Les images de salons et d'employés sont stockées sur le système de fichiers serveur avec un mapping dynamique pour l'affichage web.
